# 🎬 Movie Review Sentiment Analysis (NLP Project)

This project is an end-to-end Natural Language Processing (NLP) application that predicts whether a movie review is positive or negative using Machine Learning.

---

## 🚀 Project Overview
This project demonstrates:
- Text preprocessing
- Feature engineering using TF-IDF
- Machine learning classification
- Model evaluation
- Deployment with Streamlit

The model was trained on the IMDb movie review dataset.

---

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression
- Streamlit
- Joblib

---

## 📊 Model Performance
- Accuracy: **88.9%**
- Balanced precision and recall
- Evaluation using classification report and confusion matrix.

---

## 💡 Features
- Predicts sentiment of movie reviews
- Confidence score output
- Interactive web interface
- End-to-end ML pipeline
- Model saved and reused for deployment.

---

## 🌐 Web App
The project includes a Streamlit web application where users can input their own reviews and get real-time predictions.
